Botones

Este es el componente bloque donde definimos las propiedades de nuestro botón y el cual extendemos en los modificadores. 

%btn {
    display: inline-block;
    line-height: $baselineheight * 2;
    color: $button-color;
    text-align: center;
    vertical-align: middle;
    font-size: fs(normal);
    white-space: nowrap;
    background-color: $button-background;
    border-top: $button-border-width solid $button-border-color;
    border-right: $button-border-width solid $button-border-color;
    border-bottom: $button-border-width solid $button-border-color;
    border-left: $button-border-width solid $button-border-color;
    cursor: pointer;
    margin: 0.5em;
    padding-top: 0;
    padding-bottom: 0;
    padding-right: 1em;
    padding-left: 1em;
    border-radius: $radius;
    will-change: background-color, box-shadow;
    @include transition(all $transition-hover ease-in);
    box-shadow: shadow(low);
    
    &:hover {
        text-decoration: none;
        color: $button-color;
        box-shadow: shadow(middle);
    }
    &:active,
    &:focus{
        outline:none;
    }
    &:visited {
        color: $button-color;
    }
}

Todos los botones heredan del bloque %btn. Después añadimos las modificaciones: .btn--modificador. 

<button class="btn--secondary">button</Secondary>
<button class="btn--primary">Primary</button>

Link buttons

a.btn--secondary a.btn--primary 
<a href="#" class="btn--primary">btn--primary</a>
<a href="#" class="btn--secondary">btn--secondary</a>

Estados

a.btn--disabled 
<button class="btn--primary" disabled>Primary</button>
<a href="#" class="btn--primary  btn--disabled">a.btn--primary</a>

Botones de bloque

<button class="btn--secondary  btn--block">Secondary</button>
<button class="btn--primary  btn--block" >Primary</button>

Botones de estado

<button type="button" class="btn--info">btn--info</button>
<button type="button" class="btn--success">btn--success</button>
<button type="button" class="btn--warning">btn--warning</button>
<button type="button" class="btn--danger">btn--danger</button>

Botones - tamaños

<a href="#" class="btn--primary  btn--small">Pequeño</a>
<a href="#" class="btn--primary">Normal</a>
<a href="#" class="btn--primary  btn--large">Grande</a>
