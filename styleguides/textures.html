Sombras

Actualmente no hay clases definidas para aplicar estas sombras. Sólo están disponibles a través de la función shadow().

Los niveles disponibles son los siguientes:

  • default
  • low
  • middle
  • modal
  • dialog
.selector {
    box-shadow: shadow(low);
}

Alertas

Componente bloque donde definimos las propiedades de nuestros mensajes de alerta. Todos los mensajes de alerta heredan del bloque %alert y después se añaden los modificadores .alert--info. 

%alert {
    position: relative;
    margin: 0.5em 0;
    padding: 0.5em 1em 0.5em 1em;
    border: $alert-border-width $alert-border-style $alert-border-color;
    background-color: $alert-background-color;
    font-family: $font-text;
    font-size: fs(normal);
    color: $alert-text-color;
    border-radius: $radius;
    box-shadow: shadow(default);
    
    .alert__heading {
        margin-bottom: $baselineheight / 8;
        font-size: inherit;
        font-weight: bold;
        color: inherit;
    }
    
    .close {
        position: absolute;
        top: -2px;
        right: 2px;
        float: right;
        font-size: fs(small);
        font-weight: bold;
        line-height: 18px;
        color: inherit;
    }
    
    p,
    ol,
    ul,
    dl {
        margin-bottom: 0;
    }

    p + p {
        margin-top: 5px;
    }
}
